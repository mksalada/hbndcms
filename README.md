### ⚙️ Requirements

- Hugo Extended
  - Recommended version: `0.155.3` or newer
- Blowfish theme
  - Current version: `2.100.0`
- Netlify Identity `v1.0.0`
- Netlify Git Gateaway `v0.6.0`
